## `cra-template` 介绍

官方提供支持TypeScript的[创建Ray应用](https://developer.tuya.com/cn/ray/guide/tutorial)的模板项目。

## 快速上手

如何使用这些模板创建新应用？

示例：

```sh
npx @ray-js/create-app my-app
```
或
```sh
yarn create @ray-js/app my-app
```
或
```sh
npm init @ray-js/app my-app
```
